# Test Programs

The first six instructions to be implemented will be:

- 00E0 (clear screen)
- 1NNN (jump)
- 6XNN (set register VX)
- 7XNN (add value to register VX)
- ANNN (set index register I)
- DXNY (draw)

This is enough to run a simple program which displays the IBM logo.
